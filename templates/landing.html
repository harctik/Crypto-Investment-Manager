System Design

Clean, modular
architecture.

Every component is a separate Python module with clear responsibilities — no monolith, no spaghetti code.

⚙️

config.py

All settings loaded from .env — app, database, CoinGecko API, risk params, prediction engine, spread rules, and email alerts.

🗄️

database.py

SQLite persistence + CoinGecko API client. Handles users, portfolios, price history, watchlists, trades, notes, and price alerts.

🌐

app.py

Flask web server with 30+ REST API endpoints, SocketIO real-time broadcasting, rate limiting, caching, and session management.

💻

main.py

CLI menu interface for running milestones individually from the terminal. Useful for testing and development.

📊

milestone2

Mix calculator — Monte Carlo optimization, efficient frontier, Sharpe ratio analysis, and correlation matrices.

🧠

milestone3

Risk predictor — parallel volatility analysis, technical indicators (RSI, MACD, Bollinger), VaR/CVaR, and price predictions.

🛡️

milestone4

Spreading rules — rebalancing engine, allocation constraints, drift detection, stress testing, and Monte Carlo simulations.

📈

milestone5

Backtesting — multi-strategy comparison (MA, RSI, Bollinger, Ensemble) with equity curves and win/loss analytics.
